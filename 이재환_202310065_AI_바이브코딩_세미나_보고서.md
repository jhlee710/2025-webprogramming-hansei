## 1. 개요

### 1.1 바이브코딩(Vibe Coding)이란?

바이브코딩(Vibe Coding)은 AI 도구를 활용하여 개발자가 자연어로 의도를 전달하면 AI가 코드를 생성하고, 개발자는 이를 검토·수정하는 새로운 프로그래밍 패러다임입니다. 

> **"분위기(Vibe)를 전달하면 AI가 코드를 작성한다"**

기존의 한 줄 한 줄 직접 타이핑하는 방식에서 벗어나, 개발자와 AI가 협업하여 빠르게 결과물을 만들어내는 방식입니다.

### 1.2 프로젝트 목적

본 프로젝트는 **바이낸스(Binance) 거래소의 실시간 암호화폐 데이터를 시각화하고 기술적 분석 지표를 제공**하는 웹 애플리케이션입니다. AI 바이브코딩을 활용하여 개발하였으며, 이를 통해 생산성과 결과물의 품질을 분석합니다.

### 3.3 핵심 알고리즘 (AI 생성 코드)

#### SMA(단순이동평균) 계산
```javascript
function calcSMA(values, period){
  const out = Array(values.length).fill(null);
  let sum = 0;
  for(let i=0; i<values.length; i++){
    sum += values[i];
    if(i >= period) sum -= values[i-period];
    if(i >= period-1) out[i] = sum / period;
  }
  return out;
}
```

#### RSI(상대강도지수) 계산
```javascript
function calcRSI(values, period=14){
  // ... Wilder's Smoothing Method 적용
  // 14일간 평균 상승폭 / 평균 하락폭으로 계산
  const rs = avgGain / avgLoss;
  return 100 - (100 / (1 + rs));
}
```

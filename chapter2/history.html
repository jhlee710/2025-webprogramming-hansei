웹의 역사

월드 와이드 웹(World Wide Web, WWW)은 1989년 팀 버너스리(Tim Berners-Lee)에 의해 제안되어 1990년에 처음 구현되었습니다. 웹은 인터넷을 통해 정보를 공유하고 접근할 수 있는 시스템으로, 오늘날 우리가 사용하는 인터넷의 핵심적인 부분입니다.

주요 연혁

  • 1989년 - 팀 버너스리가 웹 개념을 제안
  • 1990년 - 최초의 웹 브라우저와 웹 서버 개발
  • 1991년 - 최초의 웹사이트 공개
  • 1993년 - 모자이크(Mosaic) 브라우저 등장, 웹 대중화 시작
  • 1994년 - W3C(World Wide Web Consortium) 설립
  • 2000년대 - 웹 2.0 시대, 사용자 참여와 소셜 미디어의 성장
  • 2010년대 이후 - 모바일 웹, HTML5, 클라우드 컴퓨팅 등 발전

웹의 구성 요소

  1. HTML - 웹 페이지의 구조를 정의
  2. URL - 웹 자원의 위치를 지정
  3. HTTP - 웹에서 정보를 주고받는 프로토콜
